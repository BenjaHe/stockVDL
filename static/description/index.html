! INFORMATION IMORTANTE

! Actions de parametrage necessaires au bon fonctionnement

Pour que les clients du web-shop puisse voir "leur compte", il faut desactiver une regle de securite sur les enregistrement : 1) mode debug 2) configuration -> securite -> Regles sur les enregistrement 3) nom de la regle : res_partner:portal/public: read access on my commercial partner -> modifier -> decocher la case "Actif". Si cette action n'est pas faite, l'application risque de planter pour tous les utilisateurs !!

Web-shop

Le web-shop interne de la Ville de Liege

Acces a un web-shop proposant les articles directement disponibles a l'Economat ou disponible chez un fournisseur officiel (fourniture sur marche stock).

Les commandes sont ensuite validees par le comptable referent et par la direction.

Renseigne la consommation budgetaire du client

Modifie le module "Account" et "Partner".

Ajoute sur la fiche du client (res.partner) les champs calcules suivant :
- le total des commandes tvac passee dans l'annee en cours,
- le budget qui lui est alloue (annee en cours)
- le solde qui lui reste (budget/commandes).
